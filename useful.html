    • Hypothesis generation vs confirmation
      • each observation can only be used for either exploration or confirmation, not both
      • observations can be used many times for exploration, but only once for confirmation
    • Data separation
      • 60% in exploration set, for automated exploration
      • 20% in query set, for manual comparison
      • 20% in test set
